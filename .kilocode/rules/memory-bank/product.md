# Product Context

## Problem Statement
Small to medium businesses need a reliable, fast, and easy-to-use Point of Sale system that doesn't rely on heavy infrastructure.

## Solution
A web-based POS application that can be run locally or hosted. It provides inventory management, sales tracking, and basic ERP functionalities.

## User Experience
- **Interface**: Clean, modern UI using Mantine components.
- **Accessibility**: Web-accessible, responsive design.
- **Workflow**: streamlined checkout process, easy inventory updates.

## Key Features
- **POS**: Quick sale processing.
- **Inventory**: Stock tracking, product management.
- **Reporting**: Sales reports (implied).
- **Architecture**: Client-server model (currently Next.js + PocketBase).
