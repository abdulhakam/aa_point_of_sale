# Tech Stack

## Frontend
- **Runtime**: Node.js (v18 recommended).
- **Framework**: Next.js 14.
- **Language**: TypeScript.
- **Styling**: Mantine v7 (CSS-in-JS/Modules).
- **Data Fetching**: TanStack Query v5.
- **Icons**: Tabler Icons.
- **Utilities**: Day.js, Moment.js (legacy?), Lodash.

## Backend
- **Engine**: PocketBase v0.20.1.
- **Database**: SQLite (Embedded).

## Development Tools
- **Package Manager**: npm.
- **Linting**: ESLint.
- **Formatting**: Prettier (implied).
- **Tables**: Mantine DataTable for complex data displays.
- **Printing**: react-to-print for report generation.

## Environment Setup
- **Frontend**: Run `npm install` in `frontend/`.
- **Backend**: Run `pocketbase.exe serve` in `backend/`.
