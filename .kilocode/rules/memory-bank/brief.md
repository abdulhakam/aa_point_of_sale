# Project Brief

## Overview
AA Point of Sale is a Point of Sale (POS) and ERP system designed to be lightweight and fast. It currently utilizes a Next.js frontend with a PocketBase backend, though there are indications of a planned migration to a custom Next.js backend with stricter authentication.

## Core Requirements
- **Performance**: Fast and lightweight.
- **Platform**: Web-based (modern browsers only).
- **Stack**: Next.js 14 (App Router), Mantine v7, PocketBase (Data).

## Project Goals
- Maintain a robust and reliable POS system.
- Possible migration from PocketBase to a custom backend (to be confirmed).
- Ensure ease of deployment and maintenance.
