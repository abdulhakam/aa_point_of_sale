# A POINT OF SALE USING POCKETBASE AND NEXT.JS
## Why PocketBase?
- It is small.
- It is lightweight.
- ~~It has it own static server so nextjs static exports shall work fine.~~
- It will be easy to migrate from if requirements need a more powerful DataBase.

## Why nextjs?
- It is fast.
- It is popular.
